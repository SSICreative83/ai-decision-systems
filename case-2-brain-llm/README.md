# LLM-Driven Multi-Modal Brain Data Integration

## Overview
This project demonstrates how large language models (LLMs) can orchestrate analysis across multi-modal neuroscience datasets.

## Key Components
- Multi-omics data integration
- LLM-based query interface
- Automated analysis pipelines

## Example Use Case
User query:
"Identify cell types associated with disease progression"

System:
→ triggers analysis  
→ generates plots  
→ produces interpretation  

## Note
This is a demonstration system using synthetic / public data.
